# ¿Como usarlo?

Clonate el repo y corré los siguientes comandos!:
* npm install
- en caso que no se instalen todas las dependencias, instalar lo siguientes puntos
* npm install express
* bower install
* npm install mongoose
* npm install oauth2-provider

# Ejecutando la aplicación

node app.js
